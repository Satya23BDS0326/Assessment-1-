Introduction to HTML

HTML (HyperText Markup Language) is used to create documents on the World Wide Web. It is simply a collection of certain key words called "Tags" that are helpful in writing the document to be displayed using a browser on Internet. HTML was developed by Tim Berners-Lee in 1992.

Features of HTML

  • HTML is easy to learn and use.
  • HTML is platform-independent.
  • HTML is used to create structured documents.
  • HTML supports multimedia elements like audio, video, and images.

Basic HTML Tags

Tag Description
<html> Defines the HTML document
<head> Contains meta information about the HTML document
<body> Defines the body of the HTML document
<p> Defines a paragraph
<a> Defines a hyperlink

Conclusion

HTML is the foundation of web development. Learning HTML is the first step towards becoming a web designer or developer. It helps in creating beautiful and structured web pages.
